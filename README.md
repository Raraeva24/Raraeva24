## Proyek Unggulan

### UMY Digital Signage Management System
Sistem manajemen konten terpusat untuk layar LCD di lingkungan kampus UMY.
- Role-based access (Superadmin & Admin)
- Pengunggahan konten lintas unit kerja
- Penjadwalan konten berbasis tanggal, jam, dan hari
- REST API untuk aplikasi display
- Stack: Laravel · PHP · MySQL · Bootstrap

### Smart Home Energy Management System (AI-Powered)
Aplikasi analisis dan rekomendasi hemat energi rumah tangga berbasis AI.
- Dual model AI: TAPAS (tabel) + LLaMA 3.1 via Groq (percakapan)
- Mode CLI dan Web
- Auto-routing pertanyaan ke model yang tepat
- Stack: Golang · HuggingFace · Groq API · HTML/CSS/JS
